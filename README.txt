README.txt Specifications


BSTree = Binary Search Tree (Árvores de Busca Binária)
AVLTree = Árvore AVL

O código foi totalmente escrito em C e em ambiente Windows usando a IDE Code Blocks.

Pode-se executar o arquivo 'BSTree.exe' para verificar o resultado da compilação do código 'BSTree.c' da Árvore de Busca Binária. Ou, para compilá-lo:
	gcc -o BSTree.exe BSTree.c

Para a árvore AVL, encontra-se um arquivo do tipo header onde estão declarados todos os métodos necessários, em conjunto com a estrutura de cada nó. Pode-se executar o arquivo AVLTree.exe para visualizar o resultado da compilação ou abrir os AVLTree.c e AVLHeader.h em uma IDE no mesmo projeto e executá-lo. Ou, para compilá-lo:
	gcc -o AVLTree.exe AVLTree.c AVLTree.h
